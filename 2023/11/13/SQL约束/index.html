约束

非空约束： NOT NULL

唯一约束： UNIQUE

主键约束： PRIMARY KEY(自增：AUTO_INCREMENT)

默认约束： DEFAULT

检查约束： CHECK

外键约束： FOREIGN KEY

方法：

ALTER TABLE 表名 ADD CONSTRAINT外键名称 FOREIGN KEY (外键字段) REFERENCES 主表名(主表字段名)

实例:

alter table emp add constraint fk_emp_dept_id foregin key(dept_id) references dept(id)

多表查询

select * from 表1，表2 ；
